<h3>DC Gradient Hero &ndash; Animated Gradient Hero for OpenCart 4</h3>
<p><strong>DC Gradient Hero</strong> is a modern hero section module for <strong>OpenCart 4</strong>, designed as an alternative to classic banners, sliders and video backgrounds.<br /> Instead of heavy layouts and rotating content, it focuses on <strong>one clear message</strong>, enhanced by a smooth animated gradient rendered on <strong>canvas</strong>.</p>
<p>The module uses <strong>granim.js</strong> as a dedicated gradient animation engine, which allows creating visually dynamic backgrounds without affecting performance or Core Web Vitals.</p>

<h3>How it works</h3>
<p>DC Gradient Hero renders the background using a single <code>&lt;canvas&gt;</code> element instead of layered <code>div</code>s.<br /> Gradient animations are created from <strong>color pairs</strong>, which allows smooth transitions between multiple visual states without using video files or heavy JavaScript logic.</p>
<p>The module supports two display modes:</p>
<ul>
<li>
<p><strong>Gradient only</strong> &ndash; clean animated background</p>
</li>
<li>
<p><strong>Gradient + image blending</strong> &ndash; image enhanced with animated color overlay</p>
</li>
</ul>
<p>If no image is selected, the module automatically falls back to gradient-only mode.</p>
